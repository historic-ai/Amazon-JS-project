
# 🛒 Amazon JS Clone

This is a frontend eCommerce project inspired by Amazon, built using vanilla JavaScript, HTML, and CSS.

## 🚀 Features

- Product listing page
- Add to Cart functionality
- Cart quantity update
- Checkout page with order summary
- Payment summary calculations
- LocalStorage integration (cart persistence)
- Modular JavaScript structure
- Unit testing using Jasmine

## 📂 Project Structure

- `data/` → Product, cart, and delivery data
- `scripts/` → Main application logic
- `styles/` → CSS files
- `pages/` → HTML pages
- `test-jasmine/` → Unit tests

## 🧠 Concepts Used

- DOM Manipulation
- ES6 Modules (import/export)
- JSON (parse & stringify)
- Event Handling
- Promises (basics)
- LocalStorage API

## 🛠️ Technologies

- JavaScript (ES6)
- HTML5
- CSS3
- Jasmine (Testing)
